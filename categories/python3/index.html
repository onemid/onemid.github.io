Cannot Use Pip3 in MacOS (zlib Dependency Problem)

8月 23, 2018 分類 python3

問題描述

在使用 pip3 時，會回應 zlib相關套件的錯誤訊息

###簡單解法

開 terminal 打

xcode-select --install

之後再裝

brew install python3

如果先裝 python3 然後再裝 xcode-select 東西的話

brew reinstall python3

就重裝 python3 吧
